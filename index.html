Aviso Importante

Esta política de privacidad se aplica específicamente a la extensión EduTextMx de Chrome y cumple con las políticas de Chrome Web Store y las regulaciones educativas mexicanas aplicables.

Información que Recopilamos

Datos que SÍ Recopilamos

  • Fragmentos de texto educativo: Extraídos de libros oficiales CONALITEG solo para propósitos educativos
  • URL fuente: Para identificar la página oficial de CONALITEG
  • Marca de tiempo: Cuando ocurre la extracción para auditoría educativa
  • Información de cuenta de usuario: Email, nombre, CURP para validación de edad
  • Datos de uso: Número de extracciones, páginas accedidas (para cumplimiento legal)

Datos que NO Recopilamos

  • Historial de navegación fuera de sitios web CONALITEG
  • Datos biométricos o identificadores únicos de dispositivo
  • Información de otras pestañas del navegador o sitios web
  • Datos personales no relacionados con uso educativo
  • Datos de ubicación más allá de validación de región mexicana

Cómo Usamos Sus Datos

Propósito Educativo Exclusivo

Todos los datos se usan exclusivamente para:

  • Facilitar extracción de contenido educativo bajo Artículo 148 de la Ley Federal del Derecho de Autor
  • Asegurar cumplimiento con límites legales de derechos de autor (máximo 750 palabras, máximo 5 páginas)
  • Proporcionar funcionalidad de biblioteca educativa personal
  • Validar uso educativo exclusivo por estudiantes mexicanos
  • Mantener registros de auditoría legal para cumplimiento regulatorio
  • Generar referencias bibliográficas APA automáticas

Compartir Datos

Partes con las que SÍ Compartimos Datos

  • Autoridades educativas: Cuando se requiere para auditorías de cumplimiento legal
  • Almacenamiento seguro en la nube: Respaldo encriptado de biblioteca educativa personal del usuario (con consentimiento explícito)
  • Sin terceros comerciales: Nunca compartimos datos para propósitos comerciales

NUNCA Compartimos Con

  • Empresas de marketing o anunciantes
  • Intermediarios comerciales de datos
  • Plataformas de redes sociales
  • Terceros no educativos

Derechos y Control del Usuario

  • Eliminar todos los datos almacenados en cualquier momento
  • Exportar biblioteca educativa personal
  • Desinstalar extensión elimina todos los datos locales
  • Acceder a registro completo de auditoría de extracciones
  • Contactar admin@edutextmx.org para solicitudes de datos

Protección de Privacidad de Menores

Protección Especial para Menores

Ya que nuestra extensión sirve a estudiantes mexicanos menores de 18 años:

  • Validación CURP asegura acceso apropiado para la edad
  • Protección extra para datos personales
  • Sin marketing o targeting comercial
  • Aplicación automática de límites de contenido educativo

Marco Legal Constitucional

Fundamento Constitucional

EduTextMx opera bajo un marco jurídico sólido que combina el derecho constitucional a la educación, el uso justo educativo protegido por la LFDA, y las mejores prácticas tecnológicas.

Base Legal Principal

  • Constitución Mexicana Artículo 3: Derecho constitucional a la educación
  • Ley Federal del Derecho de Autor Artículo 148: Uso justo educativo
  • Ley General de Derechos de Niñas, Niños y Adolescentes Artículo 57: Acceso a tecnologías educativas
  • Convención ONU sobre Derechos del Niño: Acceso a recursos educativos

Límites Legales Automáticos

Nuestro sistema aplica automáticamente límites legales de derechos de autor:

  • Máximo 750 palabras por extracción
  • Máximo 5 páginas consecutivas
  • Máximo 10% del contenido total del libro
  • Extracción solo de texto (sin procesamiento de imágenes)
